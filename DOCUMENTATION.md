# תיעוד מפורט – Prompt Engineering CLI Bot

## סקירה כללית

הפרויקט מדגים תהליך של **Prompt Engineering** – שיפור איטרטיבי של פרומפטים לשיפור דיוק מודל שפה (GPT-4o-mini) במשימת המרת שפה טבעית לפקודות CLI של Windows.

---

## ארכיטקטורה

```
משתמש → Frontend (React) → Flask API (main.py) → OpenAI GPT-4o-mini
                                    ↓
                          load_system_prompt(level=4)
                                    ↓
                          prompts/prompt4.md
```

### רכיבים

**`main.py`** – שרת Flask עם נקודת קצה אחת:
- `POST /analyze` – מקבל `prompt` (טקסט חופשי), שולח ל-GPT עם system prompt, מחזיר JSON עם פקודת CLI

**`utils/load_system_prompt.py`** – קורא קובץ `prompts/prompt{level}.md` לפי רמה

**`prompts/`** – 4 גרסאות של system prompt, כל אחת משופרת על בסיס תוצאות הערכה

---

## תהליך הפיתוח – איטרציות הפרומפט

### Prompt 1 – בסיסי
- תפקיד + דוגמאות + הנחיות פשוטות
- **בעיות שזוהו:** פקודות Linux במקום Windows, פספוס בקשות לגיטימיות, אין טיפול בפקודות מסוכנות

### Prompt 2 – שיפור דוגמאות
- הוספת דוגמה שלישית עם הקשר לא רלוונטי
- **שיפור:** המודל מתעלם מתוכן אישי ומתמקד בפעולה
- **בעיה שנותרה:** עדיין מחזיר פקודות מסוכנות

### Prompt 3 – כללי פרשנות
- הפרדה בין "כללים" ל"הנחיות פלט"
- **שיפור:** פרשנות חכמה יותר של בקשות עמומות
- **בעיה שנותרה:** אין חסימה של פקודות מסוכנות

### Prompt 4 – אבטחה ופרשנות מלאה ✅
- הוספת סעיף **אבטחה ובטיחות** עם רשימת פקודות אסורות
- הוספת דוגמה 4 (echo)
- **תוצאה:** ציון 100 על כל הקלטים בהערכה

## הערכת פרומפטים

הערכה בוצעה על 14 קלטי בדיקה לפי 4 קריטריונים:

| קריטריון | תיאור |
|----------|--------|
| פורמט עקבי | האם הפלט הוא JSON תקין עם מפתח `CLI Command` |
| תקינות תחבירית | האם הפקודה תקינה ב-Windows CMD |
| אבטחה ובטיחות | האם פקודות מסוכנות נחסמות |
| ציון סופי | ממוצע משוקלל של שלושת הקריטריונים |

### תוצאות השוואתיות

| פרומפט | ממוצע ציון סופי |
|--------|----------------|
| 1 | ~67 |
| 2 | ~75 |
| 3 | ~88 |
| 4 | ~100 |

הנתונים המלאים נמצאים בקובץ [תיעוד פרומפטים.txt](תיעוד%20פרומפטים.txt)

## הגדרות המודל

```python
model = "gpt-4o-mini"
temperature = 0        # תשובות דטרמיניסטיות
max_tokens = 500
response_format = {"type": "json_object"}
```

- `temperature=0` מבטיח עקביות בין הרצות
- `response_format=json_object` מונע תשובות שאינן JSON
